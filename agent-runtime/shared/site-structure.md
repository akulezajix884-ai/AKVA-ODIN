# Структура сайта АкваВита

Версия: 1.0
Дата: 2026-03-19

---

## Общая архитектура

Сайт представляет собой одностраничный лендинг (SPA-подход) с плавной прокруткой между секциями. Навигация по якорям. Все секции разделены SVG-волнами для создания эффекта текущей воды.

Порядок секций:

```
Header (fixed)
  |
Hero + 3D placeholder
  ~ wave ~
О воде
  ~ wave ~
Преимущества
  ~ wave ~
Воронка (пошаговая)
  ~ wave ~
Продукты
  ~ wave ~
Отзывы
  ~ wave ~
FAQ
  ~ wave ~
CTA-блок
  |
Footer
```

## 1. Header

**ID:** `#header`
**Назначение:** Постоянная точка навигации и быстрого доступа к заказу. Формирует первое доверие через контактную информацию.

### Ключевые элементы

| Элемент              | Описание                                                     |
|----------------------|--------------------------------------------------------------|
| Логотип              | SVG-логотип "АкваВита" слева. Клик — scroll to top.          |
| Навигация            | Главная, О воде, Продукты, Доставка, Контакты. Якорные ссылки. |
| Телефон              | `8 (495) 233-64-71` с иконкой phone. Кликабелен (`tel:`).     |
| Режим работы         | `08:00-22:00` мелким текстом под телефоном.                    |
| CTA-кнопка           | "Заказать воду" — `.btn-primary`, справа.                     |
| Бургер-меню (mobile) | Иконка menu/x. Открывает полноэкранное меню.                  |

### Поведение

- **Desktop:** Фиксированный сверху. Высота 72px. Фон: белый с `--shadow-sm`.
- **Scroll-эффект:** При прокрутке > 50px добавляется `backdrop-filter: blur(12px)` и `background: rgba(255,255,255,0.95)`.
- **Активная секция:** Пункт навигации подсвечивается `--color-primary-700` при попадании соответствующей секции в viewport (Intersection Observer).

### Адаптация mobile (< 768px)

- Логотип + бургер-меню + телефон-иконка (без текста).
- CTA-кнопка скрыта в шапке, присутствует в мобильном меню.
- Мобильное меню: полноэкранный оверлей с фоном `--color-primary-900`, белый текст, крупные ссылки (24px), телефоны, кнопка "Заказать".
- Анимация меню: slide-in справа, 300ms.

## 2. Hero

**ID:** `#hero`
**Назначение:** Мгновенно передать ценностное предложение. Захватить внимание. Подтолкнуть к первому действию.

### Ключевые элементы

| Элемент                     | Описание                                                          |
|-----------------------------|-------------------------------------------------------------------|
| Заголовок H1                | "Чистая вода с доставкой до двери в Одинцово"                     |
| Подзаголовок                | "Бутилированная вода 19л от 199 руб. Доставка за 2 часа. Ежедневно с 8:00 до 22:00." |
| CTA-кнопка (primary)        | "Заказать воду" — `.btn-primary`, крупная.                        |
| CTA-кнопка (secondary)      | "Смотреть каталог" — `.btn-secondary`.                            |
| Бейдж доверия               | "Более 10 000 довольных клиентов" или аналог, с иконкой.          |
| 3D-плейсхолдер              | `<div id="3d-hero-placeholder">` — область под Three.js/Spline.   |

### Разметка 3D-области

```html
<section id="hero">
  <div class="hero__content">
    <!-- текстовый контент слева -->
  </div>
  <div id="3d-hero-placeholder" class="hero__3d">
    <!-- Three.js / Spline сцена загружается сюда -->
    <!-- Fallback: статичное изображение бутыли воды с каплями -->
    <img class="hero__fallback" src="assets/hero-fallback.webp"
         alt="Бутыль чистой воды 19 литров" loading="eager" />
  </div>
</section>
```

### Layout

- **Desktop:** 2 колонки (6 + 6). Текст слева, 3D справа. Высота: 100vh минус header.
- **Фон:** `--gradient-hero` с наложением декоративных пузырьков (CSS-анимация или canvas).

### Анимации

- Заголовок: fade-up, delay 200ms.
- Подзаголовок: fade-up, delay 400ms.
- Кнопки: fade-up, delay 600ms.
- 3D-сцена: scale-in, delay 300ms.
- Фоновые пузырьки: бесконечная CSS-анимация `float-up` (translateY + opacity).

### Адаптация mobile

- Одна колонка. 3D-блок сверху (уменьшенный, высота 40vh), текст под ним.
- H1 уменьшается до 36px.
- Кнопки стекаются вертикально, полная ширина.
- Фоновые пузырьки отключаются для производительности.

## Общие технические требования

### Performance

- Lazy loading для изображений ниже fold (`loading="lazy"`).
- Hero-изображение: `loading="eager"`, preload в `<head>`.
- CSS-анимации предпочтительнее JS (используют GPU: transform, opacity).
- 3D-сцена (Three.js/Spline) загружается асинхронно, не блокирует рендер.
- Шрифты: `font-display: swap`, preconnect к Google Fonts.

### SEO

- Семантическая разметка: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`.
- Все изображения с `alt`.
- H1 — единственный на странице (в Hero).

### Accessibility

- Контраст текста: минимум 4.5:1 (WCAG AA).
- Фокусные кольца на интерактивных элементах.
- ARIA-атрибуты: `aria-expanded` для аккордеона, `aria-label` для кнопок с иконками.
- Skip-to-content ссылка.
- Клавиатурная навигация для слайдера и аккордеона.
